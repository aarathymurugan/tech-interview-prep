# Find Locations of Even Numbers

Write a function that takes in a list of numbers. It should return the
**indices** of the even numbers.

For example:

```
[2, 5, 3, 4] => [0, 3]

[1, 3, 5] => []

[0, 2, 4, 6] => [0, 1, 2, 3]
```
